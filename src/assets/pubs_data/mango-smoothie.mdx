# How to make a mango smoothie at home

### Ingredients

-   Mango
-   Milk (non-diary substitutes are fine)
-   Cinnamon
-   Grass jelly
-   Honey
-   Ice

### Equipment

-   MIPS-based food processor meanswhile this is some `inline code int()`

### Procedure

1. Put all ingredients in processor
2. Turn processor on full throttle
3. Consume with pleasure :)
